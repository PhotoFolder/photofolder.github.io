Prevent lock-in by simply using files

PhotoFolder takes advantage of open formats to give the features of Google/Apple Photos without the lock-in.

Get Started



PhotoFolder stores all in a regular folder

Location , faces and objects are not in a separate database. They are stored in the file's metadata

Sync with any cloud provider or on a USB drive . They are just files!





Advance Search

PhotoFolder detects faces and objects in images, and writes this information in the metadata. This enables complex search capabilities while ensuring 100% privacy.




Map and timeline

Use the map to find pictures geographically, or search by date.
