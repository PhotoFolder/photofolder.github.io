Prevent lock-in by simply using files

PhotoFolder takes advantage of open formats to give the features of Google/Apple Photos without the lock-in.

Get Started



PhotoFolder stores all the information in a regular folder.

Metadata such as location, faces and objects are stored in the file itself, not in a separate database.

You can sync this data with any cloud provider or store it on a USB drive. They are simply files!
